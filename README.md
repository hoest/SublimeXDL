SublimeXDL
==========
Als je "[Package Control](https://sublime.wbond.net/)" hebt geÃ¯nstalleerd in SublimeText en je wilt handigheidjes bij XDL's, doe dan het volgende:

`CTRL+SHIFT+P` => `pcaddrepo` => 'Package Control: Add repository' => `<ENTER>`

Plak in het invoer-veld deze URL: `https://github.com/hoest/SublimeXDL`

Installeer na het uitvoeren van deze actie het pakket "SublimeXDL" via de normale wijze.

Nu heb je wat code-completions voor in je XDL's.
